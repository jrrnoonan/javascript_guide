Assignment Operators

Assignment Operators assign values to JavaScript variables. Think of them as short-hand algebra formulas.

Operator Example Same As
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y
&= x &= y x = x & y
^= x ^= y x = x ^ y
|= x |= y x = x | y
<<= x <<= y x = x << y
>>= x >>= y x = x >> y
>>>= x >>= y x = x >> y
