JavaScript Events

HTML events are "things" that happen in HTML elements. JavaScript can coded to react to these events

HTML events can something done by either the browser or the user. For example,

  • A HTML page finished loading
  • A HTML input field was changed
  • A HTML button was clicked

The simplifed way of writing "event code" is described below. You canuse single or double quotes

<element event='some JavaScript'>

<element event="some JavaScript">

Click the button below to iniate the JavaScript event that changes this text!! Example of text below.

<button onclick="document.getElementById('button').innerHTML = 'You Clicked The Button'">

Notice 'button' after getElementById. The paragraph changed by clicking the button has its id set to button.

Event Description
onchange A HTML element has been changed
onclick The user clicks a HTML element
onmouseover The user moves the mouse over a HTML element
onmouseout The user moves the mouse away from a HTML element
onkeydown The user pushed a key down
onload The browser has finished loading the page

What are JavaScript Events useful for

JavaScript event handlers are useful to handle and verify user input, actions, and browser actions:

  • Things that should be done everytime a page loads
  • Things that should be done when the page is closed
  • Content that should be verified when a user inputs data
  • More

Many different methods can be employed to let JavaScript work with events:

  • HTML event attributes can execute JavaScript code directly
  • HTML event attributes can call JavaScript functions
  • You can assign your own event handler functions to HTML elements
  • You can prevent events from being sent or handled
  • Much more
