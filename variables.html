JavaScript Variables

JavaScript variables are containers for storing values.Var is the keyword for variables

var x= 5;
var y = 6; var z = x + y;

All JavaScript variables must be given unique names, which are known as identifiers Identifiers can be short names like x or y, or longer more descriptive names like CreditCardNumber. The general rules for contructing variables names (identifiers) is listed below:

  • Names can contain letters, digits, underscores and dollar signs ($).
  • Names must begin with a letter, dollar sign ($) or underscore (_)
  • Names are case sensitive (x and X are different variables
  • Reserved words (JavaScript keywords) cannot be used

It is important to note the in JavaScript = is an assignment operator, not an equal to operator. For example, the following does not make sense in algebra, but makes sense in JavaScript.

x = x + 5

Digits and Strings

There are many value types in JavaScript. For now we will focus on strings (text) and digits(numbers). Numbers don't need to be sourrounded with quotation marks ("") but strings do. If you put some digits in quotes, the rest will be treated as strings. For example, the following will return 234.

var x = "2" + 3 + 4;

One Statement Many Variables

Multiple variables can be included in one statement.

var person = "Johnny", carName = "Hyundai", price = 300;
