JavaScript

JavaScript is one of the 3 languages all developes must learn, along with HTML, and CSS. HTML defines the content of webpages, CSS styles it, while JavaScript is used to program the behaviour of webpages. Outside of websites, JavaScript is occasionally used for desktop, server, and database apps.

The Script Tag

In HTML, all code must be inserted between the opening <script> and closing </script> tags.

JavaScript Functions and Events

A JavaScript function is a block of JavaScript code, that can be executed when "called" for. For example, a script could be called for by clicking a button.

JavaScript in <body>

Javascript can be placed inside the <body> section of a HTML page. Placing scripts at the bottom of a body element improves display speed, because it script compilations slows down the display. Placing script at the bottom of the body also makes neater code.

External JavaScript

Javascript can be placed in external files. This is pratical when the same code is needed in many different web pages. Other advantages of external JavaScript sheets include:

  • Seperating HTML and code, making it easier to read and maintain
  • Cached JavaScript files can speed up page loads

JavaScript uses .js as its file extension.

To use an external script, put the name of the script file in the src (source) attribute of a <script> tag.

<script src="myScript.js"></script>

<script src="https://www.w3schools.com/js/myScript1.js"></script>

Do not wrap external Javascript in a <script> tag. This is already done in the HTML tag. Javascripts in external JavaScripts are seperated by semicolons (;).
