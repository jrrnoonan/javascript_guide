JavaScript Data Types

In order to operate on variables, JavsScript neesds to know the type of data its working with. JavaScript variables hold many types of data, including numbers, strings, and objects. These different variables are known as data types (many programming languages use data types). Without data types, a computer can't accurately solve this:

var x = 16 + "Tuesday";

Usually, we don't add numbers to text(strings), so how does JavaScript handle this? Push the buton below to find out.

As displayed in the alert box, JavaScript treats numbers as strings when they are added to strings

JavaScript evaluates expressions from right to left. Different sequences can produce different results: For example:
var x = 16 + 4 + "Chicken"; will compute 20Chicken.
var x = "Lettuce" + 16 + 4 will compute Lettuce164

These above statements are treated differently because of the order of strings and numbers. JavaScript treats all numbers as numbers until it reaches a string. At that point, the sum of all previous numbers is treated as a string, and all proceeding numbers are treated as strings.
