JavaScript Syntax

JavaScript syntax is the set of rules of how JavaScript Programs are constructed: 


    var x, y;           //How to declare variables
    x = 5; y = 6;       //How to assign values      
    z = x + y;          //How to compute values

JavaScript Values

JavaScript has 2 types of values: fixed and variable. These variables are either numbers, or strings (text).

Fixed values are called literals. Variable Values are called variables.

JavaScript Literals

Numbers

Numbers can be written with or without decimals
5415.379
53412.

Strings

Strings are text, written within double or single quotes
"John Doe"
'John Doe'

JavaScript Variables

In programming languages, variables are used to store data values.

JavaScript uses the var keyword to declare variables.

An equal sign is used to assign values to variables.

In this example, x is declared as a variable, then assigned the value 6:


        var x;

        x = 6;

JavaScript Operators

JavaScript uses arithmetic operators (+ - * /) to compute values:

(5 + 6) * 10

JavaScript uses an assign,emnt operator (=) to assign values to variables

var x, y;
x = 5;
y = 6;

JavaScript Expressions

A JavaScript expression is a combination of values, variables, and operators which compute to a value. This compuration is called an evualation. For 4 * 7 computes to 28, while the computation of x * 7 will depend on the value of x.

The values can be numbers or strings. For example, "John" +" "+"Doe" evualates to "John Doe".

JavaScript Keywords

JavaScript keywords are used to identify actions to be performed. For example, the var keyword tells the browser to creaste variables.

var x, y;
x = 5 + 6; y = x * 10;

Comments

Code after double slashes or between /* and */ will be treated as a comment. Comments are ignored and will not be executed. This will be expanded upon more later.

JavaScript Identifiers

In JavaScript, identifiers are used to name variables, keywords, functions, and labels. These rules for legal names are similar im most programming languages. The first character must be a letter, an underscore, or a dollar sign ($). Subsequent characters may be letters, digits, underscores, or dollar signs.

Case Sensitive

All JavaScript identifiers are case sensitive. The variables Firstname and firstname are two different variables.

Joining Multiple Words in One Variable

Programmers often use hyphens (-) to join multiple words into one variable. This is not allowed in JavaScript as hyphens are reserved for the subtraction symbol. Instead, programmers usually use capitialize the first letter of a new word, or use underscores (_).
