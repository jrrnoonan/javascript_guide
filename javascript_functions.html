JavaScript Functions

JavaScript function is a block of code designed to perform a particular task, and is invoked when something calls it. The example below returns a paragraph that simply says 12, the product of 4 * 3. This particular function was involded by the getElementById script. 


        <!DOCTYPE HTML>
        <head>
        <script>
        function ,myFunction() {
            document.getElementById("ex.1").innerHTML = 5+6;
        }
        </script>
        </head>
        <body>

        <p id="ex.1"></p>

        <script>
        function myFunction(p1, p2) {
            return p1 * p2;
        }
        
        document.getElementById("ex.1").innerHTML = myFunction(4, 3);
        </script>
        </body>
        </html>

Lets break down the parts of the function above.It begins with the function keyword, followed by the name (function names may be letters, digits, underscores, and $ signs). The function above is called my function.

Next comes the parimiters, that are set inside the parentheses (). This function's parameters are p1 and p2. The arguments and values of p1 and p2 come when the function is invoked (p1 and p2 are defines as 4 and 3, and they are multiplied together.)

Why Functions

Functions allow you to reuse code (define the code once and use it many times). You can also use the same code with different arguments to produce different results. For example, the function below is used multiple times to return convert different temperatures to Celcius from Fahrenheit. 


        <!DOCTYPE HTML>
        <head>
        
        </head>
        <body>

        <p id="ex.1"></p>

        <script>
        function toCelcuis(f) {
            return (5/9) * (f-32);
        }

        document.getElementById("ex.1").innerHTML = toCelcuis(77);
        </script>
        </body>
        </html>

Functions Used as Variable Values

Functions can also be used the same way as a variable.

var x = toCelsius(77);

var text = "The temperature is " + x + " Celsius";

var text = "The temperature is " + toCelsius(77) + "Celsius";

Local Variables

Variable declared within a JavaScript function become LOCAL to he function. Local variables can only be accessed from the function. Becasue they can only be accessed from within a function, variables with the same name can be used within different functions.
